# Completed Plans Archive

Store finished execution plans here as immutable records.

## Naming
Use a stable, sortable filename format:
- `YYYY-MM-DD-<short-title>.md`

## Minimum Archive Content
- Final objective and outcome
- What changed
- Validation evidence summary
- Residual risks / debt links
- Follow-up actions

## Process
1. Mark active plan as done.
2. Copy/move finalized plan into this folder.
3. Add links to related debt or runbook updates.
4. Ensure current work starts from `docs/exec-plans/active/_TEMPLATE.md`.
